# Contributing

## Setup

```sh
git clone https://github.com/leomosley/the-real-contributions-graph.git
cd the-real-contributions-graph
bun install
```

## Development

Work off the `dev` branch. Open PRs targeting `dev`, not `main`.

```sh
git checkout dev
git checkout -b your-feature
```

## Commits

Follow [Conventional Commits](https://www.conventionalcommits.org).

```
feat: add something new
fix: correct a bug
chore: boring but necessary
```

## PRs

Keep them small and focused. CI must pass before merging.
